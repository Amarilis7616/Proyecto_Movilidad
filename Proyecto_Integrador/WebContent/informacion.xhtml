Información:

El siguiente programa va realizar registros de las personas mediante su movilización,
describiendo el cada lugar con su porcentaje desde el comienzo de su recorrido,
tambien muestra cada fecha, incluyendo reportes, entre otros. El programa
contiene el código fuente realizado en Java con xhtml, donde la ventana
principal contiene el nombre de los autores que lo realizaron, para
continuar esta la página de iniciar sesión donde ingresa el administrador,
y puede registrar usuarios,cambiar la clave, bloquear usuarios si ya no exiten,
como tambien puede consultar todos los usuarios que estan dentro de este programa.
Se realiza también una base multidimencional donde se van almacenar los datos
con cada uno de los reportes que van a ser presentados en esta aplicación web.
Los reportes deberán ser para contestar las preguntas que cada grupo tendrá en
el archivo de requerimientos del Datawarehouse. Para subir este proyecto se
lo hizo creando una cuenta en GitHub creando un repositorio con el cual trabajamos
los 3 integrantes añadiendo avances cada día.
